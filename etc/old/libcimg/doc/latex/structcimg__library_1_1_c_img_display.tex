\subsection{Detailed Description}
This class represents a window which can display \hyperlink{structcimg__library_1_1_c_img}{CImg} images and handles mouse and keyboard events. Creating a {\ttfamily \hyperlink{structcimg__library_1_1_c_img_display}{CImgDisplay}} instance opens a window that can be used to display a {\ttfamily CImg$<$T$>$} image of a {\ttfamily CImgList$<$T$>$} image list inside. When a display is created, associated window events (such as mouse motion, keyboard and window size changes) are handled and can be easily detected by testing specific {\ttfamily \hyperlink{structcimg__library_1_1_c_img_display}{CImgDisplay}} data fields. See cimg\_\-displays for a complete tutorial on using the {\ttfamily \hyperlink{structcimg__library_1_1_c_img_display}{CImgDisplay}} class.
