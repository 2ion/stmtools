\subsection{Detailed Description}
Class used to compute basic statistics on pixel values of a \hyperlink{structcimg__library_1_1_c_img}{CImg} image. Constructing a \hyperlink{structcimg__library_1_1_c_img_stats}{CImgStats} instance from an image CImg$<$T$>$ or a list CImgList$<$T$>$ will compute the minimum, maximum and average pixel values of the input object. Optionally, the variance of the pixel values can be computed. Coordinates of the pixels whose values are minimum and maximum are also stored. The example below shows how to use \hyperlink{structcimg__library_1_1_c_img_stats}{CImgStats} objects to retrieve simple statistics of an image : 
\begin{DoxyCode}
      const CImg<float> img("my_image.jpg");                 // Read JPEG image f
      ile.
      const CImgStats stats(img);                            // Compute basic sta
      tistics on the image.
      stats.print("My statistics");                          // Display statistic
      s.
      std::printf("Max-Min = %lf",stats.max-stats.min);      // Compute the diffe
      rence between extremum values.
\end{DoxyCode}


Note that statistics are computed by considering the set of {\itshape scalar\/} values of the image pixels. No vector-\/valued statistics are computed.
